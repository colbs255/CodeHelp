Loops

In this section, you will start to get a better idea of the power of computing. As you probably know, computers can repeatedly do calculations or basic actions really quickly, and you are about to learn how to make them do that. It is done through something called a “loop”. A loop essentially repeats a block of code until some kind of end condition is met. A basic case for a loop would be a factorial (Ex: 4 factorial = 4! = 4 x 3 x 2 x 1 = 24). You could put a number like 5 into the loop, and then the loop will subtract 1 from 5 to get 4 and multiply that by 5, then subtract 1 from 4 to get 3 and repeat. The main types of loops are “for loops” and “while loops”. There is also a special kind of loop called the “enhanced for loop” which we will learn in the next lesson, and the “do while loop” which isn’t used as often as a normal while loop.

while loop

A while loop looks like this:

while (testCondition) {
    //if test condition is true then this code will run
}

A while loop is essentially an if statement that keeps repeating. The code tests if something is true, and if it is true then the code block will run, and then it will test the condition again. If the condition is true again then code will run again, if not then the program will move on. So in English: “While this condition is true, do this. If the condition is false, move on.” Here is an example with factorials.

int factorialNum = 6;
int answer = 1;
while (factorialNum > 0) {
    answer *= factorialNum;
    factorialNum--;
}
System.out.println("Answer: " + answer);

This code calculates “6!”. It starts out by checking if factorialNum is greater than 0, and since 6 is greater than 0, the code in the brackets runs. The program multiplies 1 by 6 then subtracts 1 from 6 to get 5 and then repeats, eventually forming the answer.

for loop

A for loop is like a condensed version of a while loop. Here is the general syntax:

for (initiating condition; end condition; do this after each iteration) {
    //code
}

So in the while loop example, a variable called factorialNum was created and the while loop kept modifying it and checking if it was greater than 0. A for loop does all of this in one line. Here is the factorial code as a for loop:

int answer = 1;
for (int factorialNum = 6; factorialNum > 0; factorialNum--) {
    answer *= factorialNum;
}

The first parameter (int factorialNum = 6) in the for loop is ran only when the for loop starts, the second parameter (factorialNum > 0) is tested at the beginning of each iteration, and the third parameter is ran after each iteration of the for loop

Notice that there are only 2 semicolons in the parameters of a for loop. You do not put a third one at the end. Also make sure you realize how this is equivalent to the code in the while loop example. For loops are usually the most common loops that you will see since they are essentially shorthand for while loops.

do while loop

General sytnax:

do {
    //code
} while(testCondition);

A do while loop works the exact same way as a while loop except the code in the loop will always run at least one time. One iteration will occur within the loop and then the test condition will be checked. So you should only use while loops when you are certain that you want the code within the while loop to occur at least once. With that being said, do while loops are not that common; usually while loops or for loops are used

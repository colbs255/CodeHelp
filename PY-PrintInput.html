Print and Input

Fun Stuff

Now that you know about variables, you can learn how to design useful programs for actual users. To do this, you would have to save the input of a user. For example, if you wanted to calculate the surface area of a sphere, you would have to ask the user for the radius of the sphere and then perform the necessary calculations.

Getting Input

Getting input from the user is fairly easy in python the syntax looks like this varName = input(“Message Text”). Essentially, the program will call the input method which will print your message to the user and wait for the user to enter a value (type the value and press enter) for your variable which the method will set equal to your variable. After getting the value, just perform the calculation for the surface area and print out your answer to the user.

pi = 3.14159265
radius = input(“Enter the radius: ”)
surfaceArea = 4 * pi * radius * radius
print(“Surface area: ” + surfaceArea)

Casting

The input method automatically treats the input data as a string, but if you want the program to check that what the user entered is a int or float, you must cast it. The syntax for casting looks like this varType(input(“message”)). This will make the compiler check if the user entered a value that falls under the desired type and will throw an error if it does not. Here are some examples

usersAge = int(input(“Enter age:  ”))
usersName = input(“Enter Name: ”)
usersWeight = float(input(“Enter weight: ”))

Practice

Practice using your newfound input and output knowledge by making programs. You could make a program that finds the average of some values, finds the midpoint of 2 lines, automates the Pythagorean theorem or whatever you want.

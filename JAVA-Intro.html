Introduction to Java

Getting Started

You have probably heard by now that Java works on just about any device. This is because Java works on a virtual machine, which is essentially a program that translates the code to run on your specific device. Activating a Java program essentially requires two steps: compiling and running. After you write the code, you compile it. This means that the computer is translating the human-readable code into byte code that your device can read. Then you just run the byte code file with the virtual machine. If this does not make sense to you now, that is okay because the whole process only requires two commands and it is not essential to understand it when you start programming.

What you need

If you only wanted to run a Java program, you would only need a Java Runtime environment (JRE), and most computers already have this. If you want to program in Java then you need a Java Developing Kit (JDK) and this includes a JRE. Just go to the Oracle JDK website to download the JDK that corresponds to your system. You should also get a text editor or an Integrated Development Environment (IDE) to actually write the code. Text editors are very light-weight, fast, and versatile. An IDE is basically a high powered text editor that can compile and run the program for you, and they will automatically find errors in your code and suggest how to fix them. You decide what is best for you, a text editor will require you to figure out every mistake for yourself and you will probably understand the language faster, but it will be more frustrating and not as efficient for writing code in general. Many college professors only allow text editors when you are learning how to program because they require you to find your own errors and understand what you are doing. I would recommend starting with a text editor and then moving on to an IDE when you are more comfortable with the language. The best text editors are Sublime and Atom (do not just use Notepad on windows or textEdit on mac, they are not optimized for programming) and the best Java IDEs are Eclipse and IntelliJ (I prefer IntelliJ). You can google one of these and you will find the downloads page with instructions.

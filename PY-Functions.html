Functions

A function is a block of code that is given a name and can be called wherever you want in a program (methods in java). They are useful when you need to do the same thing repeatedly within your code. You have actually been using built in functions within Python already. print(), pow(), len(), input(), and all of your other math functions are functions.

Calling Functions

When you call a function, you type its name followed by parenthesis with all of the parameters that it needs within in the parentheses separated by commas. The pow() function requires two parameters: the base and the exponent. Whenever a function is called, control is handed over to the function, and then returned to the rest of the program when the function ends.

Creating Functions

When creating a function, you must consider what values the function will take in, what it will do with those values, and what value it will return. Here’s the syntax of a basic function:

def functionName(parameters):
    #function code body
    return varOrExpression

Functions always start with the keyword def followed by the function name with the parameters in parentheses. The parameters are the values that you pass into the function to use in the method body, and the return statement specifies the value that the function will give back to the user. The round function for example takes in any kind of number, performs some operations in the body, and then returns the rounded version of the number. Here’s an example of a function that converts degrees Celsius to degrees Fahrenheit.

def toFahrenheit(celsius):
    return celsius * 1.8 + 32

This was pretty simple right? Just use the keyword “def”, name the method “toFahrenheit” take in a variable called Celsius and return an expression that converts degrees Celsius to degrees Fahrenheit.

Local Variables

Variables and parameters in functions are called local variables, meaning they are local to the method, and belong to the method, not the rest of the python file. They are recreated each time the function is called and deleted each time the function ends. You create local variables by initializing the variables inside of method definition.

Global Variables

Global variables are variables that can be called from anywhere within the python file, so they can be called within a method or outside of a method. You create global variables by initializing them outside of the function. Here is an example that shows local and global variables.

cityTempC = 30 # this is a global variable
def toFahrenheit(celsius):
    # celsius here is local variable
    fahrenheit = celsius * 1.8 + 32 #fahrenheit is also a local variable since it was created in a method
    return fahrenheit
print(toFahrenheit(cityTempC))

Exercises

Make a method called printUser that takes in 2 strings: firstName and lastName, and an int called age. Return one string that has the form “[firstName] [lastName] is [age] years old.”

Answer

def printUser(firstName, lastName, age):
    return (firstName + “ ” + lastName “ is ” + age “ years old.”

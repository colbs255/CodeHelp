Loops

In this section, you will start to get a better idea of the power of computing. As you probably know, computers can repeatedly do calculations or basic actions quickly, and you are about to learn how to make them do that. It is done through something called a “loop”. A loop essentially repeats a block of code until some kind of end condition is met. A basic case for a loop would be a factorial (Ex: 4 factorial = 4! = 4 x 3 x 2 x 1 = 24). You could put a number like 5 into the loop, and then the loop will subtract 1 from 5 to get 4 and multiply that by 5, then subtract 1 from 4 to get 3 and repeat. The main types of loops are “for loops” and “while loops”.

while loop

A while loop has this general syntax:

while testCondition:
    #if test condition true, this code will run and then repeat

A while loop is essentially an if statement that keeps repeating. The code tests if something is true, and if it is true then the code block will run, and then it will test the condition again. If the condition is true again then code will run again, if not then the program will move on. So in English: “While this condition is true, do this. If the condition is false, move on.” Here is an example with factorials.

factorialNum = 6
answer = 1
while factorialNum > 0:
    answer *= factorialNum
    factorialNum -= 1
print("Answer:", answer)

This code calculates “6!”. It starts out by checking if factorialNum is greater than 0, and since 6 is greater than 0, the code in the brackets runs. The program multiplies 1 by 6 then subtracts 1 from 6 to get 5 and then repeats, eventually forming the answer.

for loops

A for loop is like a condensed version of a while loop. Here is the general syntax:

for iteratingVar in sequence:
    #do this code

For loops are useful for iterating through compound data structures like lists, tuples, and dictionaries. Take a look at these examples:

name = "Ron Swanson"
for letter in name:
    print(letter)

During each iteration of the for loop, the loop essentially sets the iterating variable equal to each value of the set of data it is iterating. Strings can be iterated by a for loop. In this example, a for loop iterates through each character of the name string, causing each character of the string to be printed out one by one on each line. 

testScores = [92, 92, 88, 90, 88]
mySum = 0
for x in testScores:
    mySum += x
print(mySum / len(testScores))

In this example, the for loop iterates through each grade in the tescScores list and adds each value to the mySum variable. The program then divides the new value of mySum by the length of the testScores array (number of elements in the list) to get the grade average.

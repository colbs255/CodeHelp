Print/Input

Fun Stuff

Now that you know about variables, you can learn how to design useful programs for actual users. To do this, you will have to save the input of a user. For example, if you wanted to design a program that finds the area of circle for a user, you would have to ask the user for a radius and then save the users response.

Making a Scanner

In Java, you use a scanner object to take in user input. Creating a scanner object requires 2 steps in Java.

Step 1: import your scanner class. Java has certain built in classes that you have to import from a package into your project to use. Importing the scanner requires only one statement that you put above your class declaration: import Java.util.Scanner;.

import java.util.Scanner;
// see the import ^^
public class ScannerExample {
    public static void main(String[] args) {
        
    }
}

Step 2: Create a scanner object from the Scanner class you imported. We will learn more about objects later (I know I keep saying you will learn certain things later, but for now these things really wouldn’t be useful to you and probably wouldn’t make sense), but the general syntax for creating an object is as follows objectType objectName = new objectType(Parameter);. In the case of a scanner object, you will type Scanner input = new Scanner(System.in);. This creates a new scanner object named “input”, it could be named anything but most people usually name it “input” or “keyboard”.

import java.util.Scanner;
public class ScannerExample {
    public static vod main(String[] args) {
    Scanner input = new Scanner(System.in);
    
    }
}

Using a Scanner

Now you can use the scanner object to take in a user input. Usually you print out a question or prompt letting the user know what you want them to enter and use your scanner object like this.

import java.util.Scanner;
public class CircleArea {
    public static vod main(String[] args) {
    Scanner input = new Scanner(System.in);
    
    System.out.println("Enter the raidus of the circle");
    double radius = input.nextDouble();
    
    System.out.println("Area of circle: " + (radius * radius * 3.14));
    }
}

You already know most of this. The only new thing is the nextInt(). This is called a method of your scanner object. A method is essentially a function or action (you will learn more about methods soon), and this method accepts your input from the keyboard as an int. You use a different method for each type of variable you want to get. Here are some other methods...

  • nextInt() : gets integer
  • nextDouble() : gets double
  • nextChar() : gets char
  • nextLine() : gets a String

***Note that nextString() is not a method. You use nextLine().

Printing Methods (functions)

You already know System.out.println(). Now you will learn two more (they are pretty easy). They are System.out.printf() and System.out.print().

print() is the same as println() except you it does not make a new line when it is called. For example, if you did System.out.print(5) on one line and then System.out.print(2) on the other then the console would print 52 not 5 on one line and 2 on the next like when you use println().

printf() (print format) is pretty versatile and mainly used for printing out specific number of decimal places. You probably will not use it that often, but it is good to know. Here is an example, System.out.printf(“%.2d”, 3.1415);. This will print pi to 2 decimal places. You put the % sign in the statement to denote where you want the variable to go, then put a number after a decimal to denote how man decimal places you want, and the d tells the program that it is a double.

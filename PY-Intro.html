Introduction to Python

Python is different from languages like Java or C++ because it uses an interpreter rather than a compiler to run. This means that you can run Python code as soon as you create it rather than compiling it first because the interpreter essentially reads through your code line by line. Another nice thing is that Python does not require code to be in a class or method to run, this makes it much easier for beginners to start programming and can make creating simple programs much faster.

What you need

Many operating systems now have Python installed by default. If you’re on a mac and want to check if you have Python installed then press cmd + space and type “ter” then select the terminal application. Then type python into the terminal and press enter. To do this on Windows go to “command prompt” and type Python. If you see something like “Python 2.7.10” then you have Python. This tutorial however teaches you the syntax of python 3 not 2, so if you only have 2 then should install it from Python’s website. You should also get a text editor or an Integrated Development Environment (IDE) to actually write the code. Text editors are very light-weight, fast, and versatile. An IDE is basically a high powered text editor that can compile and run the program for you, and they will automatically find errors in your code and suggest how to fix them. You decide what is best for you, a text editor will require you to figure out every mistake for yourself and you will probably understand the language faster, but it will be more frustrating and not as efficient for writing code in general. Many college professors only allow text editors when you are learning how to program because they require you to find your own errors and understand what you are doing. I would recommend starting with a text editor and then moving on to an IDE when you are more comfortable with the language. The best text editors are Sublime and Atom (do not just use Notepad on windows or textEdit on mac, they are not optimized for programming) and the best Java IDEs are Eclipse and IntelliJ (I prefer IntelliJ). You can google one of these and you will find the downloads page with instructions.

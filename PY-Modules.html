Modules

In Python, you can get functionality and data of other Python programs by importing modules and packages. A module is essentially just a python file, and you can use modules from different places to reduce the amount of coding you have to do. Importing a module looks like this:

import moduleName

A good example of a module is the Python math module. You would import it like this

import math

Now you can use various features from the math module rather than recoding the features from scratch like this

math.functionName()

Here is an example of the math module being used

import math
math.ceil(3.4) #returns 4, the smallest integer greater than x (rounds up)
math.factorial(4) #returns 24, the factorial of x
math.pow(2,3) # returns 8, x to the power of y
math.pi #variable for pi
math.e #variable for e
math.sin(math.pi) #returns 0, the sin of x radians (there are also tan and cos functions)

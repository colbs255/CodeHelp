Classes and Objects

Now you can be introduced to Object Oriented Programming (OOP). Object Oriented Programming is about creating your own types of data (objects) and manipulating them in your own specified way. This makes modelling things in the real world much easier and organized. While Python is not exactly an object-oriented language, it does support object-oriented programming. Python can utilize classes and objects. A class is a guideline (often called blueprint) for the structure of something (an object). A class defines what an object can do, what can be done to an object, and what sort of attributes an object has. An object is the actual instantiation of the class.

Letâ€™s say you wanted to make a video game with airplanes. You could make a class that defines what an airplane is in your videogame, giving it properties such as top speed, current speed, and a name by declaring variables for these attributes. You could also define its actions like taking off, changing speed, and landing. You would then instantiate your class by making a plane object that you can manipulate in the game. The general syntax of a class looks like this:

class ClassName:
    attribute1Name = data
    attribute2Name  = data
	
    def function1(params):
        #function1 body
    def function2(params):
        #function2 body

Notice how just like methods and control statements, everything that belongs to a class must be indented to the right one level. Here is an example of an airplane class would look like:

class Airplane:
    topSpeed = 200
    currentSpeed = 0
    name = "Spruce Goose"
    
    def takeOff(self):
        print("We are taking off now")
    def changeSpeed(self, speedChange):
        self.currentSpeed += speedChange
        if self.currentSpeed > self.topSpeed:
            self.currentSpeed = self.topSpeed
    def land(self):
        print("We are landing")

Instantiating Classes and using Objects

Instantiating a class or creating an object looks like this:

objectName = className(parameters)

The parameters are passed into the constructor which you will learn in the next part of this lesson. In the airplane example, you would instantiate an airplane like this:

myPlane = Airplane()

Constructors

Congrats, you have now learned how to create classes and instantiate objects from a class, but what if you want each object to have unique attributes when you create them? You could then give each object unique attributes, like a name and top speed in the airplane example. You can do that in Python with constructors. A constructor is a special function that is called every time you instantiate an object, and has this general syntax:

def __init__(self, paramaters):
    #constructor body

The self is a variable for the object itself. Whenever you make a function inside a class, self should be the first parameter. Python automatically passes in self, so you dont have to pass in self when you call the function on the object. Whenever you are referencing variables that belong to the object, you must put self. before the variable like in the changeSpeed function. Also, keep in mind that there are two underscores on each side of the init keyword not one. A constructor for the airplane class would look like this:

def __init__(self, top, current, name):
        self.topSpeed = top
        self.current = current
        self.name = name

This will take in top, current, and name as parameters, and then set the variables that belong to the airplane object equal to these parameters. This allows you to create unique airplane objects. You would initialize an object of this type like this:

airplane1 = Airplane(570, 0, "Boeing 747")

So the final code for the Airplane class looks like this:

class Airplane:
    topSpeed = 200
    currentSpeed = 0
    name = "Spruce Goose"

    def __init__(self, top, current, name):
        self.topSpeed = top
        self.current = current
        self.name = name

    def takeOff(self):
        print("We are taking off now")
    def changeSpeed(self, speedChange):
        self.currentSpeed += speedChange
        if self.currentSpeed > self.topSpeed:
            self.currentSpeed = self.topSpeed
    def land(self):
        print("We are landing")

Calling Functions

You can now call a functions on the object like this:

airplane1.takeOff()
airplane1.changeSpeed(200)
airplane1.land()

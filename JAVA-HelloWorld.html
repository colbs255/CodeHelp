Hello World

Getting Started

We are about to write our first program. If you are using a text editor, then click File > New (cmd + N on mac or ctrl + N on windows) then File > Save (cmd + S on mac or ctrl + S on windows) and name it HelloWorld.java. You can save the program anywhere you want, but remember where you save it.

Your First Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
        // prints hello world
    }
}

If you are using an IDE, then just press the run button (should look like a play button) and move on to the next section. If you are using a text editor, then running the program is a little more complicated because you need to use terminal or command prompt. On a mac, press cmd + space and type “ter” then select the terminal application. Now you have to navigate to the directory where your file is located. Find the folder that your file exists it then type cd on the terminal and drag the folder into your terminal. This should put the path to your file into your terminal and look something like this cd ~/Users/username/Documents. You have just told the terminal to navigate to the directory your class is in. Now you can compile your java file. Type javac and then the name of your file like this javac HelloWorld.java. If you type ls you should see a list of the files in your directory and “HelloWorld.class” should be one of them. This is the compiled version of your code. Now type java HelloWorld and your program should run. If you are on Windows, go to the “command prompt” and then do the same thing but replace the ls command with dir. Practice doing this! You will be doing this every time you run your code.

What does this code mean?

Your Java code will always go inside of a class. We will go into more detail about classes, but for now just think of a class as a container for your code. Your class should always be named the same thing as the name of the file. In this example, your class name is “HelloWorld” and your file name is “HelloWorld.java”.

This line contains many words that are important to Java and it is okay if they do not make sense initially. For now, just know that this line denotes the header of your main method. A method is essentially a function or a block of code, and the main method is the first method that your program will run as soon as you run the program.

Your main code is the System.out.println(). This is a special method that will print text to your terminal whenever you call it. In this case, your text is “Hello world”. Notice the text is inside of inside of quotation marks. If you do not use quotation marks, then the code will not work. This is because text is represented as something called a String in Java. We will discuss strings a lot more soon.

Notice that the println() statement ends with a semicolon. All Java statements end in a semicolon. The semicolon tells Java that this is the end of the statement. Java does not care if you put code on a different line, it only looks at the semicolons.

The // prints hello world part is called a comment. A comment is something that you can type into the program, and the compiler will never see or use it. They are very handy for notes in your code.

Does this seem hard?

If this seems complicated right now, that is perfectly normal. All of these concepts will make a lot more sense very soon. The point of this is just to show you how to make a working program and get you excited about programming

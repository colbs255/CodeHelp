Math Functions

Python has many built in math tools that make programming easy and intuitive. All of these tools belong to the math module. A module is like a class in java which is just a file with functions and variables that someone else can use. Many modules must be imported into your code (you will learn this soon), but the math module is actually imported by default. Here are some of the most useful functions from the math module.

Function Input Output Examples
abs(x) Number Input's absolute value (number's distance from 0 abs(-2.3) -> 2.3
hex(x) int Input's value in lowercase hexadecimal form hex(48) -> '0x30'
len(str) Object Returns length of input(often string) len("Jane") -> 4
max(x1, x2) Number(s) Largest value of inputs max(2,3,-1,0) -> 2
min(x1, x2) Number(s) Smallest value of inputs min(2, 3, -1, 0) -> -1
pow(x, y) Numbers Vale of x to the y power ppw(2, 3) -> 8
round(x) Number Round input to the nearest whole number round(2.5) -> 3

Assignment Shortcuts

You will often change the value of variables while programming. Let’s say you just wanted to add the number 1 to your variable. One way is like this varName = VarName + 1. This is valid in Python because it evaluates the expression before assigning it to the variable. This code gets the value of the variable, adds 1 to the variable, and then assigns the variable that new value. A shortcut that you will see often in Python to doing this is varName++;. This is a shortcut that will add 1 to your variable. It is 100% equivalent to varName = VarName + 1. It is just a nice feature that the creators of Python added.

Let’s say you wanted to add a number that isn’t necessarily 1 to your variable. There is a shortcut for that also that looks like this varName += 2, you can use this for whatever number you want.

There is also a shortcut for multiplication that looks like this varName *= anyNumber. This will multiply your variable by the number you enter and set it equal to that new value in the process.

One more thing to note is that ++varName and varName++ are actually both valid actions in Python, but they work differently (this distinction is commonly tested in college CS courses and interviews). The distinction can be seen when you pass them into the print method. Let’s say you have an int called myNumber that equals 4. If you typed print(myNumber++), what will be printed out? 4 or 5 because the code is adding 1 to myNumber? This code will print 4, because myNumber++ will return the myNumber value before adding 1 to it. What if you typed print(++myNumber)? This would print 5, because ++myNumber will add 1 to the variable and then return it. So make sure you know the difference between these 2 forms and know when you should use them.
